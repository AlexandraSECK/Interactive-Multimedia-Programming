Definitions :

HTML : (HyperText Markup Language) It's a programming language used to create webpages. It formates and organizes a webpage thanks to tags that indicates how should be the page and what are the elements ( title, paragraphe, image etc) and how it is linked to other pages. It is usually used with other languages like CSS or JavaScript.

CSS : Cascading StyleSheets. It is a style sheet language. The principle of CSS is to have in a document the visual caracteristics of webpage. Then the document is called into the HTML document ( can be other document like XML for example) and the style is applied to it. It used the tags so each element can have its own style. But classes can also be defined.

Javascript : It is a object-oriented programming language that can be added into a HTML document ( but it can also be used to code extension for browser). It permits to create scripts that are interpretated by the web browser. JavaScript can be directly in the HTML document or in a extern document and its aim is to dynamiser the webpage by create interaction with the user (with functions for example) or animation ( ex: diaporama).

HTTP: (HyperText Transfer Protocol), it is a protocole used in the internet, it permits to tranfer files ( like HTML docs, images etc) between a browser and a webserver. thanks to a URL. The web browser will send a request to a webserver to have the files.

HTML5 : This is the actual version of HTML that is used. It's this version that made the actual organisation (ex : heading content) it introduce new API like tags for audio and video, which reduce the usage of javascript or libraries.

Dynamic HTML (DHTML) : It is a set of internet technologies that permits to have HTML pages more interactive. It is composed of HMTL CSS and Javascript.
